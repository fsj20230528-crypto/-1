# 浙江美食图片印章扩展设计

## 目标

为“打卡系统 -> 打卡城市美食”增加可扩展的图片印章能力。

- 先为整套杭州美食生成统一风格的图片印章
- 当前范围包含 `西湖醋鱼`、`龙井虾仁`、`东坡肉`、`片儿川`、`定胜糕`、`葱包桧`
- 后续用户继续以“图片 + 美食名”的形式补充或替换杭州美食
- 现已扩展为所有打卡城市默认生成同风格圆章 logo

## 方案

### 数据结构

- 在美食数据项上新增可选字段，例如 `logoImage`
- 当条目存在 `logoImage` 时，渲染图片印章
- 当条目不存在 `logoImage` 时，按统一 prompt 模板自动生成圆章图片
- 仅在没有显式图片且自动生成失败时，继续沿用现有 SVG 图标逻辑

### 展示规则

- 保持当前集章卡片版式不变
- 图片印章在现有图标槽位中显示
- 图片采用圆形裁切
- 使用 `object-fit: cover` 保持比例稳定
- 统一使用圆章母版风格：
  - 浅米色底
  - 青绿色边框
  - 西湖烟雨 / 远山 / 湖面作为背景母题
  - 主体菜品居中偏大
  - 水彩 + 掐丝珐琅感

### 范围控制

- 保留杭州已手工确认过的图片链接
- 其他城市按统一母版风格自动扩展，不需要逐条手写图片链接
- 后续若用户对某道菜有指定图片，仍可用 `logoImage` 单独覆盖

## 验收标准

- 所有打卡城市的美食条目都能显示同风格圆章图片
- 杭州现有 6 个美食条目保留专属图片效果
- 后续新增或替换任何城市的美食图片时，不需要再改渲染结构
