# 登录背景原图入库设计

## 目标

将用户提供的原始木船照片真正纳入仓库管理，并替换当前登录左侧品牌区使用的示意图背景。

## 方案

- 将原图以静态资源形式保存到仓库，例如 `assets/login-brand-boat.jpg`
- 根目录 `index.html` 改为引用仓库内图片路径
- 保留现有视觉处理：
  - 品牌区限定使用
  - 提亮
  - 降低视觉压强
  - 叠加浅色雾化遮罩
- 同步更新 `zheli-xunwei/dist/` 中对应的静态产物，保持仓库内容一致

## 资源规则

- 提交原图本体到 git，而不是外链
- 文件名稳定、语义明确，便于后续替换
- 页面样式只负责视觉处理，不再依赖在线生成图

## 验收标准

- 仓库中可见原图文件
- 登录左侧背景使用仓库内原图
- 预览中可确认背景替换成功
- 根目录页面与 `dist` 产物保持一致
